# tazebao wordpress theme – digital interface template

## description

this repository contains a custom wordpress theme developed within the project “tazebao – cultural heritage active innovation for sustainable society” (pe00000020, cup j33c22002850006).

the theme provides the frontend interface for the public exploration of cultural heritage collections managed through the tazebao semantic repository plugin.

it is designed as a reusable and adaptable template for museums, archives and cultural institutions interested in publishing digital collections through accessible and narrative-driven web interfaces.

## scientific and cultural context

the theme was developed in the context of research and experimentation in:
- digital museology
- narrative interfaces for cultural heritage
- accessible and inclusive web design

it has been tested on the art brut collections of the university museums of Turin, including the Museo “Lombroso” and the Museo di antropologia ed etnografia.

## main features

the theme provides:

- responsive and accessible frontend layout
- multiple visualization modes for digital collections (grid, carousel, table)
- item detail pages with semantic metadata
- support for narrative paths and curated sequences
- integration with multimedia resources (images, video, 3d models)
- multilingual-ready structure
- compliance-oriented design for public institutions

the theme does not manage data directly, but consumes content exposed via api by the tazebao semantic repository plugin.

## technical requirements

- wordpress (open source cms)
- php 8.x
- active installation of the tazebao semantic repository plugin
- standard wordpress rest api enabled

no proprietary frameworks or services are required.

## accessibility

accessibility is a core design principle of the theme.

the layout and interaction patterns are designed to:
- comply with wcag 2.1 aa guidelines
- follow italian agid accessibility recommendations
- support different navigation preferences and user abilities

## reuse and customization

the theme is released as open source to encourage:
- reuse in other cultural heritage projects
- visual and functional customization
- adaptation to different collections and institutional identities

it can be extended through child themes or custom plugins without modifying the core codebase.

## relation to other components

this theme represents the public interface layer of the tazebao ecosystem.

it relies on:
- the tazebao semantic repository plugin for data modeling and content exposure
- external or internal api endpoints for synchronization of collections and paths

other applications and installations may consume the same repository using different frontends.
